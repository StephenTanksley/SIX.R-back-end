# Back End Documentation

## Dependencies

- Express
- CORS
- Helmet
- Knex
- SQLite3
- Supertest
- Jest
- Nodemon
- Dotenv
- Cross-env
- BcryptJS

## Download Project and Install Dependencies

- fork and clone this repository.
- **CD into the folder** where you downloaded the repository.
- run `npm i` to download all dependencies.
